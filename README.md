# Girnari Export India 🧅

**Premium Onion Powder, Exported with Integrity.**

Welcome to the official repository/codebase for the Girnari Export website! Girnari Export is a leading manufacturer, supplier, and exporter of premium dehydrated onions, garlic, ginger, and other vegetable powders and spices from Gujarat, India.

## 🌟 About the Project

This project contains the source code for the Girnari Export website, showcasing our premium agricultural products, our global export capabilities, and our commitment to quality. The website is built with a modern, responsive design to serve our international clientele effectively.

### Key Offerings Showcased
- **Premium Varieties:** White, Red, and Pink onion powders rooted in heritage.
- **Private Labels:** Custom packaging and sourcing tailored to global visions.
- **Bulk Export:** Expertly sourced and graded bulk onion powder trusted by brands worldwide.
- **Other Products:** Dehydrated Garlic, Ginger, Fried Onions/Garlic, Vegetable & Fruit Powders, Indian Spices, and Instant Soup Premixes.

## 🏆 Certifications & Quality Standards

We are continuously committed to global quality standards. Every batch is independently tested and certified.
- **FSSAI** (Food Safety and Standards Authority of India)
- **APEDA** (Agricultural and Processed Food Products Export Development Authority)
- **SPICES BOARD INDIA**

## 💻 Tech Stack
- **HTML5** with semantic structure and SEO optimization (JSON-LD, Open Graph)
- **CSS3** with modern features (Flexbox, Grid, CSS Variables) for a fully responsive and rich design
- **Vanilla JavaScript** for dynamic interactions (Hero Slider, Mobile Navigation, Counters)
- **SVG Sprites** for scalable, lightweight iconography

## ✨ Project Highlights
- **SEO Optimized:** Implements best practices for search engine visibility.
- **Performance Focused:** Uses modern web image formats (`.webp`), lazy loading, and efficient CSS/JS delivery.
- **Responsive Design:** Ensures a seamless experience across mobile, tablet, and desktop devices.
- **Dynamic Interactions:** Engaging micro-animations, smooth scrolling, and an interactive hero slider.

## 🚀 Getting Started

To view the website locally:
1. Clone this repository.
2. Open `index.html` in your preferred web browser.
3. For the best experience, you can serve it through a local development server (e.g., VS Code Live Server).
